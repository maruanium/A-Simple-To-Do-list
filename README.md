
# TODO List (C)

A simple command-line TODO list written in C.

## Features

- Add tasks
- List tasks
- Mark tasks as completed
- Mard tasks as uncompleted (under dev)
- Delete tasks
- Edit tasks
- Save tasks to a file (done!)
- Load tasks from a file (done!)
- TUI Interactive (under dev)(probablly will take some time)
## Build && run

```bash
make && ./todo
```
